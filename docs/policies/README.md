# ポリシードキュメント集

このディレクトリでは、運用・品質に関する横断的なポリシーを管理します。各ファイルは AGENTS.md および `docs/README.md` から辿れるようにしてください。

- `project-basics.md`: プロジェクト前提と秘密情報の扱い
- `development-principles.md`: 開発方針と品質原則、他ドキュメントへの入口
- `approval.md`: 実行・承認モデルと反映ガード
- `documentation.md`: ドキュメント配置ポリシーと命名ルール

新しいポリシーを追加する際は、本 README に概要を追記し、関連する上位ドキュメントからの導線を整備してください。
