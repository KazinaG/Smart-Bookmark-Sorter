# プロダクト方針

このディレクトリでは、開発再開時の判断軸となる高レベル方針を管理します。

- `high-level-direction.md`: プロダクトの目的、優先順位、品質方針
- `file-structure-policy.md`: 現状構造の課題、構造方針、受け入れ条件
- `mv3-minimal-startup-spec-delta.md`: 現状動作確認を可能にする最小 MV3 起動化の仕様差分
- `mv3-event-driven-processing-spec-delta.md`: Service Worker 前提のイベント駆動化と状態永続化の仕様差分
- `ui-modernization-jquery-removal-spec-delta.md`: jQuery依存除去とUI最小モダン化の仕様差分

実装に先立って方針を更新する場合は、`提案 → 明示承認 → 反映` を守ります。
